# ஜாதகம் — இலவச Deployment வழிமுறை (Vercel)

இந்த project-ல் இரண்டு பகுதிகள்:
- `public/index.html` — பயனர் பார்க்கும் வெப் பக்கம் (Frontend)
- `api/horoscope.js` — உங்கள் Client ID/Secret-ஐ பாதுகாப்பாக வைத்திருந்து ProKerala API-ஐ அழைக்கும் Serverless Function (Backend)

**முக்கியம்:** உங்கள் Client Secret எங்கும் `public/` கோப்புகளில் சேர்க்கப்படவில்லை. அது Vercel Environment Variable-ஆக மட்டுமே சேமிக்கப்படும்.

## படி 1: GitHub-ல் பதிவேற்றவும்

1. https://github.com -ல் இலவச கணக்கு உருவாக்கவும் (ஏற்கனவே இல்லையெனில்).
2. புதிய repository உருவாக்கவும் (எ.கா. `jathagam-app`).
3. இந்த முழு `jathagam-app` கோப்புறையையும் அந்த repository-க்கு பதிவேற்றவும் (upload / git push).

## படி 2: Vercel-ல் Deploy செய்யவும் (இலவசம், Credit Card தேவையில்லை)

1. https://vercel.com -க்குச் சென்று "Sign Up" செய்யவும் — உங்கள் GitHub கணக்கைக் கொண்டு உள்நுழையவும்.
2. Dashboard-ல் **"Add New" → "Project"** என்பதைக் கிளிக் செய்யவும்.
3. நீங்கள் பதிவேற்றிய `jathagam-app` repository-ஐத் தேர்ந்தெடுத்து **Import** செய்யவும்.
4. Framework Preset "Other" எனக் காட்டும் — அப்படியே விடவும்.
5. **"Environment Variables"** பகுதியில் இவற்றைச் சேர்க்கவும் (பெயர்களை அப்படியே copy செய்யவும், மதிப்புகளாக உங்கள் ProKerala Client ID / Secret-ஐ paste செய்யவும் — இவற்றை GitHub repository-ல் ஒருபோதும் சேர்க்க வேண்டாம்):
   - `PROKERALA_CLIENT_ID`
   - `PROKERALA_CLIENT_SECRET`
6. **Deploy** பொத்தானை அழுத்தவும். சில நிமிடங்களில் உங்கள் தளம் தயார்!
7. Vercel தரும் URL (எ.கா. `jathagam-app.vercel.app`) மூலம் யாரும் உங்கள் தளத்தை பயன்படுத்தலாம்.

## படி 3: சோதனை

தளத்தில் பெயர், பிறந்த ஊர், தேதி, நேரம் உள்ளிட்டு "ஜாதகம் கணிக்க" பொத்தானை அழுத்தவும். நட்சத்திரம், ராசி, லக்னம், ராசி சக்கரம் ஆகியவை காட்டப்படும்.

## குறிப்புகள்

- **இலவச வரம்பு**: ProKerala-வின் இலவச திட்டம் மாத வரம்பு/நிமிட வரம்பு கொண்டது. அதிக பயன்பாடு இருந்தால் ProKerala Dashboard-ல் உங்கள் plan-ஐ சரிபார்க்கவும்.
- **இடம் கண்டறிதல்**: பிறந்த ஊர் பெயரை OpenStreetMap (Nominatim) மூலம் அட்சரேகை/தீர்க்கரேகையாக மாற்றுகிறோம். ஊர் பெயர் சரியாக இல்லையெனில் பிழை காட்டப்படும் — மாவட்டம்/மாநிலம் சேர்த்து முயற்சிக்கவும் (எ.கா. "திருச்சி, தமிழ்நாடு").
- **நேர மண்டலம்**: இயல்பாக இந்தியா (+05:30) எனக் கணக்கிடப்படுகிறது. வெளிநாட்டில் பிறந்தவர்களுக்கு "மேம்பட்ட அமைப்புகள்" பகுதியில் சரியான UTC Offset-ஐ மாற்றவும்.
- **Client Secret பாதுகாப்பு**: இந்த Secret-ஐ யாருடனும் பகிர வேண்டாம் (GitHub-ல் பொதுவெளியில் இட்ட repository-ல் கூட சேர்க்க வேண்டாம் — Environment Variable-ஆக மட்டுமே Vercel dashboard-ல் வைக்கவும்).
